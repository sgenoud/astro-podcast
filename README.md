# Astro Starter Kit: Podcasts

```sh
npm create astro@latest -- --template sgenoud/astro-podcast
```

This is a basic template to create a podcast site. You will need to upload your
mp3 on a CDN (Cloudflare R2 might be a nice fit).

The aim for this is to be easy to hack to make it your own.

You can have a look at the [demo site](https://astro-podcast.netlify.app/)

All commands are run from the root of the project, from a terminal:

| Command                       | Action                                             |
| :---------------------------- | :------------------------------------------------- |
| `npm install`                 | Installs dependencies                              |
| `npm run dev`                 | Starts local dev server at `localhost:4321`        |
| `npm run build`               | Build your production site to `./dist/`            |
| `npm run preview`             | Preview your build locally, before deploying       |
| `npm run astro ...`           | Run CLI commands like `astro add`, `astro check`   |
| `npm run astro -- --help`     | Get help using the Astro CLI                       |
| `npm run mp3-info a-file.mp3` | Get the information necessary to add a new episode |
